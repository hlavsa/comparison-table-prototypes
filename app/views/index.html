CAIT Comparison Table Prototypes

For this project, we've needed to create a table that is responsive, and has a focus on allowing you to compare various options against each other. There is a basic table in the Design System, but it is too simple for our use case. So we've been creating something more tailored.

So far, there have been three phases of iterations (of which Phase 3 was the final). These are detailed below.

Phase 3

  • Option 7 - Unresponsive Scroll Table v3.0

    After Phase 2, we had a long discussion and decided that version 2 of Option 4 was the most appropriate. That is because it preserves the ability to directly compare when on different screen sizes, which is the most important part of this table. We made a few tweaks to it though, to make it clearer that you have to scroll to see the whole table.

Phase 2

For Phase 2, we decided that Options 1 and 4 were the most appropriate for our particular use case, but that we liked some of the benefits of Option 3. So we've tried to combine the best bits of those options.

  • Option 5 - Responsive Plain Table v2.0

    In this iteration, we drew from the Design System's Tab pattern. We liked that accordions allowed you to switch between different groups of information on mobile, but found it didn't work on desktop. So this pulls across the 'Table of Contents' from the Tab component, on mobile, as a quick way of navigating between content.

  • Option 6 - Unresponsive Scroll Table v2.0

    We've kept this one broadly the same as the previous version, aside from adjusting the table headers to make them links.

Phase 1

In this phase, I took existing patterns and tables that others across government had already created and used them as a base. Reference list is at the bottom of the page.

  • Option 1 - Responsive Plain Table v1.0

    This example is responsive but doesn't have the ability to expand content out on mobile.

  • Option 2 - Responsive Details Table v1.0

    This option is responsive, and allows you to hide and expand content using the Design System details component

  • Option 3 - Responsive Accordion Table v1.0

    This table is responsive, and allows you to hide and expand content using the Design System accordion component

  • Option 4 - Unresponsive Scroll Table v1.0

    This table is responsive, and allows you to hide and expand content using the Design System accordion component
